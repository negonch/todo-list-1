# Todo list

This is a React application for displaying a list of todos.
This project was created with Vite and React. 
It displayes a page title and a list of three todo items.

## Installation

1. Clone the repository from GitHub: 'https://github.com/negonch/todo-list.git'
2. Open the project folder in Terminal: 'cd todo-list'
3. Install dependencies:  'npm install'


## Run the project

1. Start the development server: 'npm run dev'
2. Open http://localhost:5173 in your browser
